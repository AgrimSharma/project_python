ScrumDo Addons

Integrate with applications you love

Jira

Jira is an defect tracking tool for teams building great software. Track bugs about ScrumDo stories or create stories from Jira bugs.

Basecamp

Basecamp is an online project management tool. Create ScrumDo stories for any Basecamp todo item. Push ScrumDo stories to Basecamp todos.

Github

Github is one of the most popular SCM tool. Create ScrumDo stories for any open GitHub issue. Get updates of source code commits in ScrumDo newsfeed.

Harvest

Harvest is an tool to track time, Send Invoice and Run Business online. Create Harvest tasks for your ScrumDo stories. Track your time and invoice customers in Harvest.
